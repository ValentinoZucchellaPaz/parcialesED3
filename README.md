# Repo de Ejercicios - Comisión

Este repositorio está pensado para compartir ejercicios, prácticas y resoluciones de la cursada.  
La idea es mantener todo organizado y que cualquiera pueda consultar o aportar material fácilmente.

---

## Clonar el repositorio

En el repo tocar el boton `Code`, copiar la url y ejecutar los comandos:

```bash
git clone URL_DEL_REPO
cd NOMBRE_DEL_REPO
```

---

## Subir archivos y cambios

Subir una carpeta con nombre de parcial e iniciales o nombre completo
Ejemplo:

```bash
/parcial_1_VZP
/parcial_2_Valentino_Zucchella_Paz
```

Dentro de esa carpeta pueden agregar código, PDFs, apuntes, imágenes, etc.

Los cambios los agrego con una pull request, subanlos a su repo clonado y pusheen, dps entren a Github a su repo y click en el boton

```
Compare & pull request
```
